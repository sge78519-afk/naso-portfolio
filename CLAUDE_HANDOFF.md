# Naso 个人独立站交接说明

这是一个静态个人作品集网站，当前技术栈是原生 HTML / CSS / JavaScript，方便本地预览、修改和部署到 Netlify。

## 项目文件

- `site/index.html`：页面结构与文案
- `site/styles.css`：视觉样式、响应式、动效
- `site/script.js`：滚动进度、滚动显现、桌面端分屏滚动
- `site/assets/`：当前网站使用的图片素材
- `netlify.toml`：Netlify 发布配置，发布目录是 `site`

## 本地预览

在项目根目录运行：

```bash
python3 -m http.server 4173 --directory site --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:4173/
```

也可以直接打开：

```text
site/index.html
```

但建议用本地服务预览，路径和浏览器行为更接近上线状态。

## 已实现的动态效果

- 首屏背景慢速呼吸
- 首屏标题分层入场
- 细线网格背景漂移
- 滚动进度条
- 模块滚动显现
- 作品卡 hover 细节动效
- 右侧竖向 `WORK / MAIL` 快捷入口，移动端变为底部胶囊
- 桌面端鼠标滚轮分页：每滚动一次跳到下一个停靠点
- 移动端保留自然滚动
- 尊重 `prefers-reduced-motion`

## 重要设计约束

- 保持暗色、克制、东方山野、极简艺术感。
- 不要做成常见模板站、营销大卡片堆砌、亮色科技渐变站。
- 不要添加夸张弹跳、旋转、粒子爆炸等过度动效。
- 作品区需要保留可读性，不要让悬浮控件遮挡正文。
- 不要恢复或使用之前被用户否定的金属牌图片。
- 移动端优先阅读体验，不要强制分屏滚动。
